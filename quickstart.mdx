## Integra en tres pasos

Esta guia resume el camino mas corto para conectar un sistema externo con Lumos CRM.

### Paso 1: configura acceso

1. Solicita credenciales para tu integracion.
2. Guarda el token bearer en un gestor seguro de secretos.
3. Define el entorno que vas a consumir (sandbox o produccion).

<Note>
  No expongas tokens en codigo fuente, repositorios ni logs de aplicacion.
</Note>

### Paso 2: ejecuta la primera llamada

```bash
curl --request GET \
  --url "https://api.tu-entorno-lumos.com/contacts?limit=10" \
  --header "Authorization: Bearer <tu_token>" \
  --header "Content-Type: application/json"
```

Si recibes codigo 401 o 403, revisa permisos y expiracion del token antes de continuar.

### Paso 3: activa eventos webhook

Configura un endpoint HTTPS en tu sistema para recibir notificaciones de cambios relevantes y mantener tus datos sincronizados.

```json
{
  "eventType": "contact.updated",
  "occurredAt": "2026-04-12T16:30:00Z",
  "data": {
    "id": "ct_1024"
  }
}
```
