## Integra en tres pasos

Esta guia resume el camino mas corto para conectar un sistema externo con Lumos CRM.

### Paso 1: configura acceso

1. Ingresa con un usuario de rol admin del tenant.
2. Genera o recupera la API key del tenant desde `POST /integrations/api-key/generate` o `GET /integrations/api-key`.
3. Guarda la API key en tu gestor seguro de secretos.
4. Identifica el `tenantId` que usaras en todas las llamadas de integracion.

<Note>
  Solo el perfil admin puede gestionar la API key de integraciones.
</Note>

### Paso 2: ejecuta la primera llamada

```bash
curl --request GET \
  --url "https://api.tu-entorno-lumos.com/integrations/v1/accounts?tenantId=<tu_tenant_id>" \
  --header "x-api-key: <tu_api_key_de_integraciones>" \
  --header "Content-Type: application/json"
```

Si recibes codigo 401, revisa que la API key corresponda al mismo tenantId.

### Paso 3: habilita endpoints de soporte

Activa los endpoints de soporte para tus flujos operativos:

1. Obtiene o rota API key cuando cambie tu politica de seguridad.
2. Usa resumen de oportunidades e hilos para contexto comercial.
3. Resuelve grantId cuando necesites trazabilidad entre sistemas.
