## Autenticacion

Todos los endpoints requieren token bearer en el header Authorization.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```

<Tip>
  Si integras webhooks, valida firma y origen antes de procesar eventos.
</Tip>
