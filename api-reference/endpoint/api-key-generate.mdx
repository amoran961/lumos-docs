---
title: 'Generar API key del tenant'
description: 'Generacion o rotacion administrativa de la API key de integraciones'
---

## Metodo y ruta

`POST /integrations/api-key/generate`

## Autenticacion

- Header `Authorization: Bearer <jwt_admin>`

## Notas

- Solo usuarios con rol admin pueden generar o rotar la clave.
- La generacion y rotacion de la API key se gestiona via consola de administracion.
- Este endpoint existe para uso administrativo controlado.
