---
title: 'Obtener API key del tenant'
description: 'Consulta administrativa de la API key de integraciones para el tenant actual'
---

## Metodo y ruta

`GET /integrations/api-key`

## Autenticacion

- Header `Authorization: Bearer <jwt_admin>`

## Notas

- Solo usuarios con rol admin pueden consultar esta clave.
- La obtencion de la API key se gestiona via consola de administracion.
- Este endpoint existe para uso administrativo controlado.
