## Flujo recomendado

1. Define tu caso de uso y los objetos que necesitas consultar o actualizar.
2. Obtiene API key con perfil admin y usa tenantId en cada request de integracion.
3. Consume endpoints REST para operaciones principales.
4. Completa setup de credenciales y OAuth para escenarios avanzados.

<Note>
  Si migras una integracion existente, comienza por quickstart y luego valida endpoint por endpoint en la referencia.
</Note>
